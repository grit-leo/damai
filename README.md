# 新品审核智能助理 MVP

这是一个浏览器插件版 MVP，用于在现有新品审批页面上打开右侧 AI 审核侧栏。

## 当前能力

- 点击插件后在当前页面打开审核侧栏。
- 以“麦总 AI 审核助理”形式展示 Agent 工作台，支持自然语言指令。
- 自动读取页面可见文本中的商品名称、商品编码、品牌、类目、规格、供应商、采购价、京东价等字段。
- 支持自动采集商品详情图链接，用于后续调用视觉模型分析。
- 支持通过本地/内网价格采集服务自动爬取官旗到手价。
- 支持手动补录官旗到手价、全网低价、BOM 成本和费用率。
- 自动计算日销、大促、最低凑单三类场景的贡利率和贡利额。
- 自动反推建议采购价。
- 生成风险等级、审核建议、风险原因。
- 一键复制审核意见或证据摘要，粘贴回原审批系统。

## AI 助手体验

插件打开后，麦总会先自动读单，并把审核任务拆成：

```text
读单完成 -> 补价格证据 -> 生成审核意见
```

用户可以直接输入自然语言指令：

```text
帮我完整审核这单
```

麦总会自动串联执行：

- 查询官旗到手价。
- 采集详情图证据。
- 读取全网低价、BOM 和费用率口径。
- 在首屏展示结论快照，并生成风险等级、建议采购价、关键理由和三场景利润测算。

也可以使用快捷指令：

- 完整审核
- 查官旗价
- 采详情图

## 安装方式

1. 打开 Chrome 或 Edge 的扩展程序管理页面。
2. 开启开发者模式。
3. 选择“加载已解压的扩展程序”。
4. 选择本目录下的 `extension` 文件夹。
5. 打开新品审批页面，点击浏览器工具栏里的“新品审核智能助理”。

## 自动爬取说明

当前自动采集采用“插件 + 采集服务”的结构：

- 插件负责从审批页读取商品信息，并发起“采集详情图”或“自动爬取官旗价”请求。
- 详情图采集服务负责访问商品详情页，抽取主图、详情图、规格图、配方图等图片链接。
- 价格采集服务负责访问官旗商品页，解析券后价、到手价、活动价、优惠券和结构化价格字段。
- 服务返回平台、店铺、链接、图片证据、价格证据、置信度和采集时间。
- 插件展示图片清单，可复制详情图链接给视觉模型分析。

接口：

```text
POST http://127.0.0.1:8787/api/detail-images
POST http://127.0.0.1:8787/api/official-price
```

请求示例：

```json
{
  "productName": "喜崽鲜泥混合价猫餐盒湿粮猫罐头主食猫头拌饭 1kg",
  "brand": "喜崽",
  "spec": "1kg",
  "officialUrl": "http://localhost:8123/demo/official-store-page.html"
}
```

正式接入时，建议把 `server/price-crawler.js` 部署成内网服务。对抖音这类没有稳定 H5 的平台，服务侧应改为云手机 RPA：打开 App、进入商品详情页、滚动采集详情图/截图，然后把图片 URL 或截图文件地址返回给插件。插件本身不要直接保存账号或绕过平台风控。

云手机 RPA 返回建议格式：

```json
{
  "ok": true,
  "platform": "douyin",
  "shopName": "品牌官方旗舰店",
  "images": [
    {
      "url": "https://internal-storage/detail-001.png",
      "type": "detail",
      "confidence": 90,
      "evidence": "商品详情页第 1 屏截图"
    }
  ],
  "capturedAt": "2026-07-01T00:00:00.000Z"
}
```
